Proyecto: Analítica de Ventas (ejemplo)

Descripción:
Este proyecto carga un archivo CSV con las ventas de una tienda durante 7 días,
calcula estadísticas básicas y genera una gráfica simple.

Estructura:
- data/ventas.csv: datos de ejemplo
- notebooks/analisis_ventas.ipynb: análisis en Jupyter Notebook
- requirements.txt: librerías necesarias

Cómo ejecutar:
1. Crear y activar entorno virtual (.venv).
2. Instalar dependencias: pip install -r requirements.txt
3. Abrir el notebook en VS Code y ejecutar las celdas.
